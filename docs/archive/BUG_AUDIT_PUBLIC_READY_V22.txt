REVIBE MARKET V22 - PUBLIC READY BUG AUDIT

FIXED IN V22
1. Category dropdown tertutup/terpotong hero video.
   Root cause: .main-navbar memakai CSS contain: layout paint sehingga panel dropdown ter-clip area navbar.
   Fix: navbar overflow visible, contain none, z-index dropdown lebih tinggi dari hero, mobile dropdown fixed.

2. Private upload .htaccess diperkuat.
   Root cause: hanya memakai "Deny from all" yang lebih cocok Apache lama.
   Fix: ditambah Require all denied + fallback Apache lama + Options -Indexes.

3. Security headers ditambah.
   Fix: Permissions-Policy dan always nosniff.

- Semua file PHP lolos php -l, tidak ada syntax error.

- Checkout sudah pakai transaksi dan SELECT FOR UPDATE.
