# Deploy Checklist Multi-Server

- [ ] Release artifact bersih.
- [ ] `.env` production di secret manager/server.
- [ ] Managed DB siap.
- [ ] Managed Redis siap.
- [ ] R2/S3/Spaces siap.
- [ ] Migration sukses.
- [ ] Worker aktif.
- [ ] Scheduler tunggal aktif.
- [ ] Load balancer healthcheck ke `/health.php`.
- [ ] `/readiness.php` sukses setelah semua config lengkap.
